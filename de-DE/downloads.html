† Unter Windows gibt es zwei prominente Binärschnittstellen: die native (MSVC) ABI, von Visual Studio, und die GNU ABI von der GCC toolchain. Welche Version von Rust du brauchst hängt hauptsächlich davon ab mit welchen C/C++ Bibliotheken du kompatibel sein willst. Für Kompatibilität mit Software die von Visual Studio kompiliert wurde, benutze die MSVC Variante von Rust; Für Kompatibilität mit Software die mit der MinGW/MSYS2 toolchain kompiliert wurde, nutze die GNU Variante.

Die MSVC Variante von Rust benötigt zusätzlich die Microsoft Visual C++ build tools für Visual Studio 2013 oder neuer. Die einfachste Art um die build tools zu bekommen ist indem du Microsoft Visual C++ Build Tools 2015 installierst. Dieses beinhaltet die Visual C++ build tools. Alternativ kannst du Visual Studio 2015 oder Visual Studio 2013 installieren und während der Installation die "C++ tools" auswählen. Um die GNU Variante zu benutzen ist keine weitere Software notwendig.

Rusts Unterstützung für die GNU ABI ist ausgereifter und wird daher allgemein empfohlen.
